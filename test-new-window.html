Тест навигации в новом окне

Инструкции для тестирования:

  1. Откройте основное приложение в браузере
  2. Нажмите на любую ссылку, чтобы открыть её в новом окне
  3. В новом окне попробуйте нажать на ссылки внутри контента
  4. Проверьте, работает ли навигация назад/вперед в новом окне

Тестовые ссылки для проверки:

Эти ссылки должны открываться в новом окне с рабочей навигацией:

Список глав Глава 1 Контент 1

Ожидаемое поведение:

  • При клике на ссылку открывается новое окно
  • В новом окне отображается соответствующий контент
  • Ссылки внутри нового окна работают
  • Кнопки назад/вперед браузера работают в новом окне

Отладочная информация:

Откройте консоль разработчика (F12) для просмотра отладочных сообщений.

В новом окне должны появляться сообщения о навигации и доступных ID контента.
