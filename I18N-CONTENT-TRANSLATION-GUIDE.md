# Руководство по переводу контента

## Текущий статус

✅ **Завершено:**
- Структура данных для поддержки многоязычности
- Переводы для всех глав (6 глав)
- Переводы для всех подглав (~40 подглав)
- Функции для использования переводов обновлены

⏳ **В процессе:**
- Переводы для контента разделов (~36 разделов)

## Структура данных

Контент должен быть в формате:
```javascript
'1.1': {
    title: {
        ru: 'Заезд ТС на площадку',
        en: 'Vehicle Arrival at Site',
        zh: '车辆到达场地'
    },
    content: {
        ru: `<h3>Направление ТС</h3>...`,
        en: `<h3>Vehicle Direction</h3>...`,
        zh: `<h3>车辆方向</h3>...`
    }
}
```

## Как добавить переводы

1. Найдите раздел контента в `app.js` (например, `'1.1': { ... }`)
2. Замените:
   ```javascript
   title: 'Заезд ТС на площадку',
   ```
   на:
   ```javascript
   title: {
       ru: 'Заезд ТС на площадку',
       en: 'Vehicle Arrival at Site',
       zh: '车辆到达场地'
   },
   ```

3. Замените:
   ```javascript
   content: `...`
   ```
   на:
   ```javascript
   content: {
       ru: `...`,
       en: `...`,
       zh: `...`
   }
   ```

## Примечания

- Функция `getTranslatedValue()` автоматически обрабатывает как старый формат (строки), так и новый (объекты с переводами)
- При отсутствии перевода для текущего языка используется русский язык как fallback
- HTML-разметка и ссылки остаются без изменений в переводах
